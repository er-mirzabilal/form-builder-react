# form-builder-react

A complete react form builder that interfaces with a json endpoint to load and save generated forms.

Technolgies used
React js 17
Material Ui
Redux for state management

Avaialble Feilds type
Text
Select
Checkbox
Radio
Image
Video
Date
Content
Signature


Installation Guide
1) Clone the repo
2) npm install
3) npm start
